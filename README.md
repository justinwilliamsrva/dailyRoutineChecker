# dailyRoutineChecker

## User Story

```
AS A busy, goal oriented person
I WANT to keep track of my daily routines
SO THAT I can my personal and business life in order
```

## Acceptance Criteri

```
GIVEN I am using a website to keep track of my tasks

WHEN the website loads
THEN I am shown my daily tasks, with a percentage of how much is completed
WHEN I click on a task
THEN The task style changes and my percentage of completed tasks is raise
WHEN I hover over the tasks
THEN a delete button appears
WHEN i double click a task
THEN I can edit the task and save it

```
